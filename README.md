# Math Quiz Game (C++)

A console-based math quiz game built in C++ that generates random arithmetic questions with different difficulty levels and operations.

### Features

- Randomly generated math questions
- Multiple difficulty levels
- Different arithmetic operations
- Instant answer feedback
- Final score calculation
- Console-based interface

### Concepts Practiced

- Variables and Data Types
- Conditional Statements
- Loops
- Functions
- Random Number Generation
- User Input Validation
- Program Flow Control

### Technologies

- C++
- Console Application

## Purpose

This project was created as part of my programming learning journey to practice problem-solving skills, function design, random number generation, and user interaction in C++.
